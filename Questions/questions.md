## Q.1 Array Question

Convert the following array to form a desired output.

```
const array = [
    {fruit : 'Apple', colour: 'Red'},
    {fruit : 'Grapes', colour: 'Green'},
]
```

Output :

```
output : {Apple : 'Red', Grapes:'Green'}
```
